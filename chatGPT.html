Configuración inicial

Esta parte sí debe estar. Es lo que hace que ChatGPT deje de responder como asistente genérico y empiece a trabajar con contexto.

Dónde entrarFoto de perfil → Configuración → Personalización → Personalizar ChatGPT.
Qué escribirTu nombre, qué haces, a quién ayudas, qué vendes y cómo quieres que te responda.
ReglaNo escribas una biografía larga. Escribe contexto útil para que la IA trabaje mejor.
Prompt para campo “sobre mí”
Soy [tu nombre]. Trabajo en [área o tipo de negocio].

Ayudo a [cliente ideal] a conseguir [resultado principal].

Mi marca vende [producto, servicio o contenido].

Mi estilo de comunicación es [directo / cálido / profesional / educativo / estratégico].

Uso ChatGPT para crear contenido, organizar ideas, desarrollar productos digitales, mejorar mis ventas y ahorrar tiempo en tareas de negocio.
Prompt para campo “cómo responder”
Respóndeme de forma clara, directa y útil.

No uses relleno.
No inventes datos.
No me des la razón si algo está flojo.
Explícame lo que debo mejorar y dame una versión mejor.

Entrega todo en bloques fáciles de copiar y pegar.
Cuando cree contenido, usa lenguaje natural, no tono robótico.
Cuando la tarea sea compleja, organiza en pasos.
Dónde entrarConfiguración → Personalización → Memoria.
Cómo usarloDile: “Recuerda que mi marca se dirige a…” o “Guarda esto para próximas respuestas…”.
Cómo corregirDile: “Olvida…” o “Actualiza esta información…”.
No guardes información innecesaria. La memoria debe ayudar, no llenar ruido.
Un proyecto por áreaEjemplo: contenido, ventas, productos digitales, marca, clientes.
Sube documentos baseGuía de marca, ejemplos de contenido, productos, tono de voz y documentos importantes.
Inicia cada chat con contextoUsa la instrucción de abajo para activar los documentos del proyecto.
Inicio recomendado
Antes de responder, revisa los documentos, instrucciones y contexto de este proyecto. Luego responde según el objetivo específico de esta conversación.

Funciones nuevas

La idea no es explicar todo técnicamente. La idea es que la persona sepa cuándo usar cada opción.

Canvas
Para documentos largos

Úsalo cuando quieras escribir, editar o convertir una guía, PDF, landing, email largo o documento con estructura.

Comando
Abre esto en Canvas y conviértelo en un documento organizado, editable y listo para convertir en PDF. Mantén títulos claros, secciones cortas y bloques fáciles de copiar.
Imágenes
Para visualizar ideas

Úsalo para fotos, mockups, personajes, productos, branding, portadas, carruseles visuales y universos de marca.

Comando
Crea una fotografía hiperrealista editorial. Mantén continuidad visual, luz cinematográfica, textura natural, fondo limpio y una estética premium. No incluyas texto dentro de la imagen.
Búsqueda profunda
Para investigar

Úsala cuando necesitas información actualizada, análisis de mercado, tendencias, comparaciones o fuentes verificables.

Comando
Haz una investigación profunda sobre [tema]. Compara fuentes actuales, resume hallazgos clave, detecta oportunidades y entrégame recomendaciones prácticas para [objetivo].
Agente
Para tareas con pasos

Úsalo cuando quieres que ChatGPT ejecute una tarea completa: buscar, organizar, comparar, crear archivos o armar un flujo.

Comando
Actúa como agente de ejecución. Primero analiza el objetivo, luego divide la tarea en pasos, ejecuta lo que puedas directamente y entrégame el resultado final organizado.
Prompt web
Crea una página web interactiva en un solo archivo HTML con CSS y JavaScript interno.

OBJETIVO: [explica la función de la página]
ESTILO VISUAL: [describe el branding]
SECCIONES: [lista las secciones]
FUNCIONES: botones para copiar prompts, acordeones desplegables, diseño responsive mobile-first.

Entrégame el código completo listo para pegar en index.html y subir a Vercel.
Prompt documento/PDF
Convierte esta información en una guía clara y profesional. Organízala con portada, índice, secciones, ejemplos prácticos, prompts copiables y cierre. El objetivo es que luego pueda exportarse como PDF.

Vocabulario visual

Estas palabras ayudan a que ChatGPT entienda más rápido qué tipo de resultado visual quieres.

Prompts listos

Bloques rápidos para copiar y pegar. En una página interactiva, esto es lo que más valor percibido genera.

Universo visual
Crea un universo visual completo para [marca/proyecto].

Incluye:
— estética principal
— paleta de color
— tipo de fotografía
— iluminación
— escenarios permitidos
— vestuario
— elemento ancla de marca
— palabras visuales clave
— negative prompt
— 5 ideas de imágenes iniciales

El resultado debe sentirse coherente, premium y reconocible.
Carrusel
Crea un carrusel de 7 slides sobre [tema] para [audiencia].

Estructura:
Slide 1: hook de 3 a 6 palabras
Slide 2: identificación del problema
Slide 3: revelación
Slide 4: mecanismo simple
Slide 5: prueba o ejemplo
Slide 6: puente hacia la acción
Slide 7: CTA claro

Entrega cada slide en un bloque separado listo para copiar y pegar.
Batch render
Genera un batch render de [número] imágenes para [marca/personaje/producto].

Mantén:
— misma identidad visual
— misma paleta
— misma iluminación
— mismo nivel de realismo
— continuidad entre todas las imágenes

Variaciones:
1. close-up
2. mid shot
3. wide shot
4. detalle del producto
5. escena lifestyle

No incluyas texto dentro de las imágenes.
Optimizar prompt
Toma este prompt y conviértelo en una versión profesional, específica y lista para usar:

[pega tu prompt aquí]

Mejora:
— claridad
— dirección visual
— estilo
— composición
— iluminación
— negative prompt
— formato final

Entrega solo el prompt final en un bloque listo para copiar.
